Vui lòng cài đặt FontAwesomeFX trước khi sử dụng
B1: Vào trang "https://bitbucket.org/Jerady/fontawesomefx/downloads/" download phiên bản
fontawesomefx-8.9.jar được cập nhật vào ngày 2 tháng 2 năm 2016.
B2: Trong Eclipse, click chuột phải vào tên dự án chọn Properties
B3: Chọn Java Build Path ở menu bên trái
B4: Chọn thẻ Libraries ở phía trên cùng
B5: Chọn Add External JARs và chọn file FontAwesomeFX đã tải về ở bước 1.
B6: Click OK để đóng cửa sổ Properties.

Vui lòng cài đặt file sqljdbc42.jar tương tự như bên.

Chúc các bạn sử dụng ứng dụng ngon miệng./.
